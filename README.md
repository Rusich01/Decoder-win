Щоб запустити проєкт, потрібно відкрити термінал у кореневій папці проєкту та виконати наступні команди:

1. npn i ;
2. npm run dev ;

Створений функціонал дає змогу переглядати інформацію про автомобіль. Після заповнення поля з VIN вам надається частина інформації про цей автомобіль. У вкладці Variables ви зможете дізнатися більше.

Якщо натиснути на поле Variable або Value, ви побачите лише ту інформацію, яку обрали.

Також функціонал дозволяє зберігати та видаляти три останні VIN-коди, які ви шукали.
